Projects

Enhancing Hardware Trojan Detection with Synthetic Data

My research focuses on enhancing hardware Trojan detection by addressing data scarcity and imbalance using Conditional Wasserstein GANs (CWGANs). This approach generates realistic power, voltage, and current signal profiles for both Trojan-infected and benign circuits. Baseline classification on original datasets achieved 92.98% accuracy, validating detection robustness. Synthetic datasets exhibited near-perfect accuracy, but a performance gap emerged when models trained on synthetic data were tested on original datasets, highlighting domain shift challenges. This work establishes synthetic data as a scalable and cost-effective solution for hardware security, paving the way for more robust detection systems..

Hardware Trojan Detection Using Fuzzy Logic

This project introduces a Fuzzy Logic-based approach for Hardware Trojan detection, addressing challenges of uncertainty, noise, and variability in side-channel signals. Unlike traditional machine learning methods, which rely on strict decision boundaries, Fuzzy Logic enables flexible and adaptive classification of power, voltage, and current signal variations, enhancing Trojan detection accuracy. The system employs a Fuzzy Inference System (FIS), utilizing membership functions to categorize signals into benign or Trojan-infected states based on their degree of deviation from normal behavior. The adaptability of Fuzzy Logic makes it effective in handling ambiguous or overlapping signal patterns, leading to robust detection even in complex real-world scenarios.

AI Agent for Trojan Detection in Side-Channel Analysisy

This project develops an AI-powered intelligent agent for Hardware Trojan detection in side-channel analysis, leveraging LLMs (LLaMA API) to analyze and classify signals from electronic circuits. The AI agent is designed to automate the detection of Trojans in hardware devices by learning patterns from power, voltage, and current signal data and identifying anomalies indicative of malicious modifications. By integrating machine learning, deep learning, and large language models (LLMs), this system enhances hardware security through advanced AI-driven analysis. The agent is equipped with a web-based interface, allowing users to input signal data, visualize detection results, and gain AI-driven insights into potential Trojan threats.
